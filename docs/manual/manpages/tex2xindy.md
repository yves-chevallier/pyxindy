## Synopsis

```text
tex2xindy [file]
```

## Description

`tex2xindy` reads from stdin a file in the `makeindex` input format
(`.idx`) and converts it to the format used by `xindy`. If `file` is
specified, statistics are written for consumption by `makeindex4(1L)`.
