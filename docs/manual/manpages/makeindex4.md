## Synopsis

```text
makeindex4 [options]...
```

## Description

`makeindex4` is a wrapper that uses `xindy` as a replacement for
`makeindex`. It aims for compatibility with `makeindex` options and
warns about unsupported ones. Requires `tex2xindy` and `xindy`.
