XINDY

A Flexible Indexing System

Roger Kehr, Institut für Theoretische Informatik, TH-Darmstadt

Tutorial, Edition March 1997 for xindy version 1.1

This is a tutorial for the xindy indexing system [5]. xindy is a general purpose index processor that can be configured for a variety of document preparation systems. We will demonstrate step by step the incremental specification of a complete index style. The examples are based on indexes that are delivered with the system so one can get acquainted with it practically.
